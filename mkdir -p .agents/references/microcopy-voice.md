# Microcopy Voice Guide (prático)

## Regras
- Use verbos específicos: “Criar pedido”, “Enviar foto”, “Aplicar desconto”
- Prefira “o que acontece agora” ao invés de “OK”
- Erro sempre com ação: “Tente novamente”, “Recarregar”, “Ver detalhes”, “Falar com suporte”
- Não culpe o usuário. Não use “inválido” sem explicar o motivo.

## Padrões prontos
### Empty state
- Título: “Nenhum X ainda”
- Texto: “Quando você fizer Y, seus X aparecem aqui.”
- CTA: “Fazer Y”

### Error state
- Título: “Não foi possível carregar X”
- Texto: “Parece que houve um problema na conexão.”
- Ações: “Tentar novamente” | “Ver status” (opcional)

### Confirmação (toast)
- “Salvo”
- “Atualizado”
- “Convite enviado”
